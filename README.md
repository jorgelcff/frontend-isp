# Desafio ISP - Frontend

Este é o frontend do projeto do Desafio ISP. Este aplicativo fornece uma interface para visualizar e gerenciar apreensões.

## Tecnologias Utilizadas

- Angular
- TypeScript
- Angular Material
- RxJS
- NgCharts
